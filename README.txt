# Bash Scripting Suite for System Maintenance

## Overview
This project automates system maintenance tasks using Bash scripts.
It includes automated backup, system update and cleanup, log monitoring, and a menu-driven interface.

## Files
1. backup.sh - Creates compressed backups of user Documents.
2. update_cleanup.sh - Updates and cleans the system.
3. log_monitor.sh - Monitors logs for critical errors.
4. maintenance_suite.sh - Menu-driven script to manage all tasks.
5. error_handling.sh - Handles errors and logs results.

## Usage
1. Make all scripts executable:
   chmod +x *.sh
2. Run the suite:
   ./error_handling.sh
